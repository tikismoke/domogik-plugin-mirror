xPL Schema
==========

The `sensor.basic <http://xplproject.org.uk/wiki/index.php?title=Schema_-_SENSOR.BASIC>`_ schema is used by this plugin.

Here are the events for which a xPL message is sent

Mir:ror faced up
----------------

.. code-Block:: none

    xpl-trig
    {
    ...
    }
    sensor.basic
    {
    device=mirror
    type=activated
    current=high
    }

Mir:ror faced down
------------------

.. code-Block:: none

    xpl-trig
    {
    ...
    }
    sensor.basic
    {
    device=mirror
    type=activated
    current=low
    }

A RFID element is detected
--------------------------

.. code-Block:: none

    xpl-trig
    {
    ...
    }
    sensor.basic
    {
    device=
    type=present
    current=high
    }

A RFID element is no more detected
----------------------------------

.. code-Block:: none

    xpl-trig
    {
    ...
    }
    sensor.basic
    {
    device=
    type=present
    current=low
    }
